#  __Nodejs course__

***
```
mkdir blog
wget https://github.com/startbootstrap/startbootstrap-clean-blog/archive/gh-pages.zip
unzip gh-pages.zip
npm init -y
npm install nodemon --save-dev
npm install express
npm install express-edge --save
```
### __Start project__
```
node index.js
```
### __start with script with package.json__

```
npm run start
```
